# grid-line-generator

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### 项目搭建参照文章

[用纯 CSS 实现网格背景](https://spacexcode.com/blog/pure-css-grid-line/)
